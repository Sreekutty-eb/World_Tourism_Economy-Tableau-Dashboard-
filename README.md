# World Tourism & Economy Analysis (Tableau Dashboard)

 # Project Overview

This project analyzes the relationship between global tourism trends and economic performance using interactive visualizations built in Tableau.

The objective is to understand how tourism contributes to economic growth by examining indicators such as:

International tourist arrivals

Tourism revenue

GDP contribution

Regional performance trends

The project provides business insights using an interactive dashboard created in Tableau.

# Objectives

Analyze global tourism trends over time

Examine tourism’s contribution to national economies

Compare tourism performance across countries and regions

Identify top-performing tourism economies

Understand the correlation between tourism revenue and GDP
